ActiveSupport::Inflector#titleize

Capitalizes all the words and replaces some characters in the string to create a nicer looking title. +titleize+ is meant for creating pretty output. It is not used in the Rails internals. +titleize+ is also aliased as +titlecase+. "man from the boondocks".titleize # => "Man From The Boondocks" "x-men: the last stand".titleize # => "X Men: The Last Stand" "TheManWithoutAPast".titleize # => "The Man Without A Past" "raiders_of_the_lost_ark".titleize # => "Raiders Of The Lost Ark"
