ActiveSupport::Multibyte::Unicode#normalize

Returns the KC normalization of the string by default. NFKC is considered the best normalization form for passing strings to databases and validations. * string - The string to perform normalization on. * form - The form you want to normalize in. Should be one of the following: :c, :kc, :d, or :kd. Default is ActiveSupport::Multibyte.default_normalization_form
