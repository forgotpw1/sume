ActionView::Helpers::OutputSafetyHelper#safe_join

This method returns a html safe string similar to what Array#join would return. All items in the array, including the supplied separator, are html escaped unless they are html safe, and the returned string is marked as html safe. safe_join(["

foo

".html_safe, "

bar

"], "
") # => "

foo

<br /><p>bar</p>" safe_join(["

foo

".html_safe, "

bar

".html_safe], "
".html_safe) # => "

foo


bar

"
