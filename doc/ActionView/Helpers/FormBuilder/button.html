ActionView::Helpers::FormBuilder#button

Add the submit button for the given form. When no value is given, it checks if the object is a new resource or not to create the proper label: <%= form_for @post do |f| %> <%= f.button %> <% end %> In the example above, if @post is a new record, it will use "Create Post" as button label, otherwise, it uses "Update Post". Those labels can be customized using I18n, under the helpers.submit key (the same as submit helper) and accept the %{model} as translation interpolation: en: helpers: submit: create: "Create a %{model}" update: "Confirm changes to %{model}" It also searches for a key specific for the given object: en: helpers: submit: post: create: "Add %{model}"
