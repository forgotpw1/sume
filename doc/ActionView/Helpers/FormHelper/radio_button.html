ActionView::Helpers::FormHelper#radio_button

Returns a radio button tag for accessing a specified attribute (identified by +method+) on an object assigned to the template (identified by +object+). If the current value of +method+ is +tag_value+ the radio button will be checked. To force the radio button to be checked pass :checked => true in the +options+ hash. You may pass HTML options there as well. # Let's say that @post.category returns "rails": radio_button("post", "category", "rails") radio_button("post", "category", "java") # => # radio_button("user", "receive_newsletter", "yes") radio_button("user", "receive_newsletter", "no") # => #
