ActionView::Helpers::FormHelper#hidden_field

Returns a hidden input tag tailored for accessing a specified attribute (identified by +method+) on an object assigned to the template (identified by +object+). Additional options on the input tag can be passed as a hash with +options+. These options will be tagged onto the HTML as an HTML element attribute as in the example shown. ==== Examples hidden_field(:signup, :pass_confirm) # => hidden_field(:post, :tag_list) # => hidden_field(:user, :token) # =>
