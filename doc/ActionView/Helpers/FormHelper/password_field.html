ActionView::Helpers::FormHelper#password_field

Returns an input tag of the "password" type tailored for accessing a specified attribute (identified by +method+) on an object assigned to the template (identified by +object+). Additional options on the input tag can be passed as a hash with +options+. These options will be tagged onto the HTML as an HTML element attribute as in the example shown. ==== Examples password_field(:login, :pass, :size => 20) # => password_field(:account, :secret, :class => "form_input", :value => @account.secret) # => password_field(:user, :password, :onchange => "if $('user[password]').length > 30 { alert('Your password needs to be shorter!'); }") # => password_field(:account, :pin, :size => 20, :class => 'form_input') # =>
