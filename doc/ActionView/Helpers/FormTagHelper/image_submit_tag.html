ActionView::Helpers::FormTagHelper#image_submit_tag

Displays an image which when clicked will submit the form. source is passed to AssetTagHelper#path_to_image ==== Options * :confirm => 'question?' - This will add a JavaScript confirm prompt with the question specified. If the user accepts, the form is processed normally, otherwise no action is taken. * :disabled - If set to true, the user will not be able to use this input. * Any other key creates standard HTML options for the tag. ==== Examples image_submit_tag("login.png") # => image_submit_tag("purchase.png", :disabled => true) # => image_submit_tag("search.png", :class => 'search_button') # => image_submit_tag("agree.png", :disabled => true, :class => "agree_disagree_button") # => image_submit_tag("save.png", :confirm => "Are you sure?") # =>
