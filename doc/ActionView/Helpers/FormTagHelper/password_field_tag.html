ActionView::Helpers::FormTagHelper#password_field_tag

Creates a password field, a masked text field that will hide the users input behind a mask character. ==== Options * :disabled - If set to true, the user will not be able to use this input. * :size - The number of visible characters that will fit in the input. * :maxlength - The maximum number of characters that the browser will allow the user to enter. * Any other key creates standard HTML attributes for the tag. ==== Examples password_field_tag 'pass' # => password_field_tag 'secret', 'Your secret here' # => password_field_tag 'masked', nil, :class => 'masked_input_field' # => password_field_tag 'token', '', :size => 15 # => password_field_tag 'key', nil, :maxlength => 16 # => password_field_tag 'confirm_pass', nil, :disabled => true # => password_field_tag 'pin', '1234', :maxlength => 4, :size => 6, :class => "pin_input" # =>
