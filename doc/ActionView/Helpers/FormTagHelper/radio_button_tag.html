ActionView::Helpers::FormTagHelper#radio_button_tag

Creates a radio button; use groups of radio buttons named the same to allow users to select from a group of options. ==== Options * :disabled - If set to true, the user will not be able to use this input. * Any other key creates standard HTML options for the tag. ==== Examples radio_button_tag 'gender', 'male' # => radio_button_tag 'receive_updates', 'no', true # => radio_button_tag 'time_slot', "3:00 p.m.", false, :disabled => true # => radio_button_tag 'color', "green", true, :class => "color_input" # =>
