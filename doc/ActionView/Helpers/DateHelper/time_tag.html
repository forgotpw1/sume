ActionView::Helpers::DateHelper#time_tag

Returns an html time tag for the given date or time. time_tag Date.today # => November 04, 2010 time_tag Time.now # => November 04, 2010 17:55 time_tag Date.yesterday, 'Yesterday' # => Yesterday time_tag Date.today, :pubdate => true # => November 04, 2010 <%= time_tag Time.now do %> Right now <% end %> # => Right now
