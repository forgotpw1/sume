ActionView::Helpers::AssetTagHelper#path_to_font

Computes the path to a font asset. Full paths from the document root will be passed through. font_path("font") # => /assets/font font_path("font.ttf") # => /assets/font.ttf font_path("dir/font.ttf") # => /assets/dir/font.ttf font_path("/dir/font.ttf") # => /dir/font.ttf font_path("http://www.example.com/dir/font.ttf") # => http://www.example.com/dir/font.ttf aliased to avoid conflicts with an font_path named route
