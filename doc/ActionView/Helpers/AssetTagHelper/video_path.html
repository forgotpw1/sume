ActionView::Helpers::AssetTagHelper#video_path

Computes the path to a video asset in the public videos directory. Full paths from the document root will be passed through. Used internally by +video_tag+ to build the video path. video_path("hd") # => /videos/hd video_path("hd.avi") # => /videos/hd.avi video_path("trailers/hd.avi") # => /videos/trailers/hd.avi video_path("/trailers/hd.avi") # => /trailers/hd.avi video_path("http://www.example.com/vid/hd.avi") # => http://www.example.com/vid/hd.avi
