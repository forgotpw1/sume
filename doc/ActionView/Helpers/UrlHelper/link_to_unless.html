ActionView::Helpers::UrlHelper#link_to_unless

Creates a link tag of the given +name+ using a URL created by the set of +options+ unless +condition+ is true, in which case only the name is returned. To specialize the default behavior (i.e., show a login link rather than just the plaintext link text), you can pass a block that accepts the name or the full argument list for +link_to_unless+. ==== Examples <%= link_to_unless(@current_user.nil?, "Reply", { :action => "reply" }) %> # If the user is logged in... # => Reply <%= link_to_unless(@current_user.nil?, "Reply", { :action => "reply" }) do |name| link_to(name, { :controller => "accounts", :action => "signup" }) end %> # If the user is logged in... # => Reply # If not... # => Reply
