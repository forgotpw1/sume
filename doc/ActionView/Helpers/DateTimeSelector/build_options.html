ActionView::Helpers::DateTimeSelector#build_options

Build select option html from date value and options. build_options(15, :start => 1, :end => 31) => " ..." If :use_two_digit_numbers => true option is passed build_options(15, :start => 1, :end => 31, :use_two_digit_numbers => true) => " ..." If :step options is passed build_options(15, :start => 1, :end => 31, :step => 2) => " ..."
