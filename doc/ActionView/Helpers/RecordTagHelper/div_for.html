ActionView::Helpers::RecordTagHelper#div_for

Produces a wrapper DIV element with id and class parameters that relate to the specified Active Record object. Usage example: <%= div_for(@person, :class => "foo") do %> <%= @person.name %> <% end %> produces:
Joe Bloggs
You can also pass an array of Active Record objects, which will then get iterated over and yield each record as an argument for the block. For example: <%= div_for(@people, :class => "foo") do |person| %> <%= person.name %> <% end %> produces:
Joe Bloggs
Jane Bloggs
