ActionView::Helpers::TextHelper#cycle

Creates a Cycle object whose _to_s_ method cycles through elements of an array every time it is called. This can be used for example, to alternate classes for table rows. You can use named cycles to allow nesting in loops. Passing a Hash as the last parameter with a :name key will create a named cycle. The default name for a cycle without a +:name+ key is "default". You can manually reset a cycle by calling reset_cycle and passing the name of the cycle.
