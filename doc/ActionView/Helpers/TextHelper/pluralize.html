ActionView::Helpers::TextHelper#pluralize

Attempts to pluralize the +singular+ word unless +count+ is 1. If +plural+ is supplied, it will use that when count is > 1, otherwise it will use the Inflector to determine the plural form. pluralize(1, 'person') # => 1 person pluralize(2, 'person') # => 2 people pluralize(3, 'person', 'users') # => 3 users pluralize(0, 'person') # => 0 people
