AbstractController::Helpers::ClassMethods#modules_for_helpers

Returns a list of modules, normalized from the acceptable kinds of helpers with the following behavior: String or Symbol:: :FooBar or "FooBar" becomes "foo_bar_helper", and "foo_bar_helper.rb" is loaded using require_dependency. Module:: No further processing After loading the appropriate files, the corresponding modules are returned. ==== Parameters * args - An array of helpers ==== Returns * Array - A normalized list of modules for the list of helpers provided.
