AbstractController::Layouts::ClassMethods#layout

Specify the layout to use for this class. If the specified layout is a: String:: the String is the template name Symbol:: call the method specified by the symbol, which will return the template name false:: There is no layout true:: raise an ArgumentError nil:: Force default layout behavior with inheritance ==== Parameters * layout - The layout to use. ==== Options (conditions) * :only - A list of actions to apply this layout to. * :except - Apply this layout to all actions but this one.
