Hash#slice

Slice a hash to include only the given keys. This is useful for limiting an options hash to valid keys before passing to a method: def search(criteria = {}) assert_valid_keys(:mass, :velocity, :time) end search(options.slice(:mass, :velocity, :time)) If you have an array of keys you want to limit to, you should splat them: valid_keys = [:mass, :velocity, :time] search(options.slice(*valid_keys))
