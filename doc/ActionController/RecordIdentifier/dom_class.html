ActionController::RecordIdentifier#dom_class

The DOM class convention is to use the singular form of an object or class. dom_class(post) # => "post" dom_class(Person) # => "person" If you need to address multiple instances of the same class in the same view, you can prefix the dom_class: dom_class(post, :edit) # => "edit_post" dom_class(Person, :edit) # => "edit_person"
