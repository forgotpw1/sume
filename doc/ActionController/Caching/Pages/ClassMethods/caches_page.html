ActionController::Caching::Pages::ClassMethods#caches_page

Caches the +actions+ using the page-caching approach that'll store the cache in a path within the page_cache_directory that matches the triggering url. You can also pass a :gzip option to override the class configuration one. # cache the index action caches_page :index # cache the index action except for JSON requests caches_page :index, :if => Proc.new { !request.format.json? } # don't gzip images caches_page :image, :gzip => false
