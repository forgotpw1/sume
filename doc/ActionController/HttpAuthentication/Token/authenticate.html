ActionController::HttpAuthentication::Token#authenticate

If token Authorization header is present, call the login procedure with the present token and options. controller - ActionController::Base instance for the current request. login_procedure - Proc to call if a token is present. The Proc should take 2 arguments: authenticate(controller) { |token, options| ... } Returns the return value of `&login_procedure` if a token is found. Returns nil if no token is found.
