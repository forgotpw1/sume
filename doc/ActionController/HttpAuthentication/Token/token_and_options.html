ActionController::HttpAuthentication::Token#token_and_options

Parses the token and options out of the token authorization header. If the header looks like this: Authorization: Token token="abc", nonce="def" Then the returned token is "abc", and the options is {:nonce => "def"} request - ActionDispatch::Request instance with the current headers. Returns an Array of [String, Hash] if a token is present. Returns nil if no token is found.
