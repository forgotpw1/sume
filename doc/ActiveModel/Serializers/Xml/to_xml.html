ActiveModel::Serializers::Xml#to_xml

Returns XML representing the model. Configuration can be passed through +options+. Without any +options+, the returned XML string will include all the model's attributes. For example: user = User.find(1) user.to_xml 1 David 16 2011-01-30T22:29:23Z The :only and :except options can be used to limit the attributes included, and work similar to the +attributes+ method. To include the result of some method calls on the model use :methods. To include associations use :include. For further documentation see activerecord/lib/active_record/serializers/xml_serializer.xml.
