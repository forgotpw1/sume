ActionDispatch::TestProcess#fixture_file_upload

Shortcut for Rack::Test::UploadedFile.new(ActionController::TestCase.fixture_path + path, type): post :change_avatar, :avatar => fixture_file_upload('/files/spongebob.png', 'image/png') To upload binary files on Windows, pass :binary as the last parameter. This will not affect other platforms: post :change_avatar, :avatar => fixture_file_upload('/files/spongebob.png', 'image/png', :binary)
