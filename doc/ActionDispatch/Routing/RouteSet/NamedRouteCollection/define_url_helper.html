ActionDispatch::Routing::RouteSet::NamedRouteCollection#define_url_helper

Create a url helper allowing ordered parameters to be associated with corresponding dynamic segments, so you can do: foo_url(bar, baz, bang) Instead of: foo_url(:bar => bar, :baz => baz, :bang => bang) Also allow options hash, so you can do: foo_url(bar, baz, bang, :sort_by => 'baz')
