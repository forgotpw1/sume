ActiveRecord::FinderMethods#first

Find the first record (or first N records if a parameter is supplied). If no order is defined it will order by primary key. Examples: Person.first # returns the first object fetched by SELECT * FROM people Person.where(["user_name = ?", user_name]).first Person.where(["user_name = :u", { :u => user_name }]).first Person.order("created_on DESC").offset(5).first
