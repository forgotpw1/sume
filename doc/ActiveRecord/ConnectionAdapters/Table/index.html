ActiveRecord::ConnectionAdapters::Table#index

Adds a new index to the table. +column_name+ can be a single Symbol, or an Array of Symbols. See SchemaStatements#add_index ====== Creating a simple index t.index(:name) ====== Creating a unique index t.index([:branch_id, :party_id], :unique => true) ====== Creating a named index t.index([:branch_id, :party_id], :unique => true, :name => 'by_branch_party')
