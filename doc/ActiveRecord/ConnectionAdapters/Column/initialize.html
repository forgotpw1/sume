ActiveRecord::ConnectionAdapters::Column#initialize

Instantiates a new column in the table. +name+ is the column's name, such as supplier_id in supplier_id int(11). +default+ is the type-casted default value, such as +new+ in sales_stage varchar(20) default 'new'. +sql_type+ is used to extract the column's length, if necessary. For example +60+ in company_name varchar(60). It will be mapped to one of the standard Rails SQL types in the type attribute. +null+ determines if this column allows +NULL+ values.
