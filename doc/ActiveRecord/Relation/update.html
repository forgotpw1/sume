ActiveRecord::Relation#update

Updates an object (or multiple objects) and saves it to the database, if validations pass. The resulting object is returned whether the object was saved successfully to the database or not. ==== Parameters * +id+ - This should be the id or an array of ids to be updated. * +attributes+ - This should be a hash of attributes or an array of hashes. ==== Examples # Updates one record Person.update(15, :user_name => 'Samuel', :group => 'expert') # Updates multiple records people = { 1 => { "first_name" => "David" }, 2 => { "first_name" => "Jeremy" } } Person.update(people.keys, people.values)
