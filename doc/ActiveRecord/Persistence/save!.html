ActiveRecord::Persistence#save!

Saves the model. If the model is new a record gets created in the database, otherwise the existing record gets updated. With save! validations always run. If any of them fail ActiveRecord::RecordInvalid gets raised. See ActiveRecord::Validations for more information. There's a series of callbacks associated with save!. If any of the before_* callbacks return +false+ the action is cancelled and save! raises ActiveRecord::RecordNotSaved. See ActiveRecord::Callbacks for further details.
