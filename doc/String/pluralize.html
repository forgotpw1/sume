String#pluralize

Returns the plural form of the word in the string. If the optional parameter +count+ is specified, the singular form will be returned if count == 1. For any other value of +count+ the plural will be returned. 'post'.pluralize # => "posts" 'octopus'.pluralize # => "octopi" 'sheep'.pluralize # => "sheep" 'words'.pluralize # => "words" 'the blue mailman'.pluralize # => "the blue mailmen" 'CamelOctopus'.pluralize # => "CamelOctopi" 'apple'.pluralize(1) # => "apple" 'apple'.pluralize(2) # => "apples"
