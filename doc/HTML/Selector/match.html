HTML::Selector#match

:call-seq: match(element, first?) => array or nil Matches an element against the selector. For a simple selector this method returns an array with the element if the element matches, nil otherwise. For a complex selector (sibling and descendant) this method returns an array with all matching elements, nil if no match is found. Use +first_only=true+ if you are only interested in the first element. For example: if selector.match(element) puts "Element is a login form" end
